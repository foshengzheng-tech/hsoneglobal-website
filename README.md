# HSONE V2 Static Website

Static GitHub Pages-ready website for HSONE.

## Deploy
1. Upload all files to a GitHub repository.
2. Enable GitHub Pages from the main branch.
3. Replace `https://example.com` in `sitemap.xml` with your real domain.
4. Replace form handling with Formspree, Netlify Forms, or your email service.

## Included
- Home
- Products: 53 product pages across 5 categories
- OEM/ODM
- Brand
- Resources
- Factory Tour
- News Center
- Support
- SEO sitemap and robots
